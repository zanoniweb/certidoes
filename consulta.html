Explicação do Web Programa

  • ✔ Passo 1: Esse web programa foi desenvolvido com as linguagens de programação: HTML, CSS, JAVASCRIPT e foi utilizado uma pastas tabela como diretório para armazenar as tabelas referentes aos dados fictícios de imóveis dos anos de 2020 até 2026. Observo que as listagens de 2020 até 2025 tem 999 inscrições imobiliarias e a de 2026 tem 1000 inscrições imobiliárias
  • ✔ Passo 2: O objetivo do programa é possibilitar um sistema rápido e prático para a consulta de metragens edificadas ou não para a finalidade de elaboração de certidões narrativas. A princípio propronho que as tabelas de dados de cada ano podem ser construídas com os arquivos da emissão dos carnes de IPTU de cada, ou seja, esse seria um banco de dados para o banco de dados web de consulta.
  • ✔ Passo 3: O sistema deverá ser consultado pelos usuários de duas formas incialmente pelas inscrições com o final "000" e depois com o final com alguma númeração que já indiquem área edificada, por exemplo "001".
  • ✔ Passo 4: A consulta individualizada por inscrição irá trazer as respectivas informações: inscrição municipal, nº da quadra, nº do lote, ano, netragem, utilização e estrutura.
  • ✔ Passo 5: Observo que a consulta nesse sistema não dispensa apuração no sistema interno da Prefeita para a finalidade de apuração se o contribuinte requereu alguma revisão histórica de IPTU.
  • ✔ Passo 6: Como exemplo coloco alguns exemplo de inscrições para teste de consulta: 1º caso: 01.001.142.0012.000 que é um terreno vago; 2º caso 01.001.142.0012.000 que é uma área com edificação; 3º caso 01.001.142.0012.002 que é uma área com o mesmo tipo de edificação; 4º caso 01.001.142.0012.999 que é uma área com edificações com metragens variadas; 5º caso 01.001.143.0012.999 que é uma área com edificações a partir de 2026;
  • ✔ Passo 7: Eu Juliano Zanoni servidor de carreira desta municipalidade mais especificamente da Receita Municipal de Cambé/Pr estou doando o código que foi criado mim e está hospedado num ambiente aberto a Receita Municipal de Cambé/Pr para habilite essa nova ferramenta ao Departamento de Cadastramento Imobiliário e com isso facilite aos atendimentos aos contribuintes de Cambé/Pr.
